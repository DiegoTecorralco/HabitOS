# HabitOS
aplicación que te ayuda a crear tus rutinas diarias,  tiene recordatorios y donde puedas ver las estadísticas, marcarlo como completado y que te lance un mensaje de felicitaciones motivándote a seguir con tus metas en el dia, notificaciones que te motiven segun las metas o recordatorios del dia, tambien puede meter metas a largo o mediano plazo
